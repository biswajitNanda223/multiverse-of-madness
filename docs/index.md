# Low-Level Design (LLD) Wiki - Index

Welcome to the LLD Wiki index. This section of the monorepo covers core concepts of Software Engineering, Object-Oriented Programming (OOP), SOLID design principles, and UML modeling schemas.

---

## 📖 Wiki Sections

### 1. [OOP Fundamentals & SOLID Principles](file:///c:/personal%20Projects/lld/docs/oop_fundamentals.md)
Learn the four pillars of OOP (Encapsulation, Abstraction, Inheritance, Polymorphism) and their Python implementations. This page also details Python magic (dunder) methods and explains all five SOLID design principles with concrete, pythonic before/after examples.

### 2. [Software Development Life Cycle (SDLC)](file:///c:/personal%20Projects/lld/docs/sdlc.md)
Understand how software products are built from initial requirements analysis to High-Level Design (HLD), Low-Level Design (LLD), automated testing, and CI/CD pipelines. This page compares Waterfall, Agile/Scrum, and DevSecOps processes.

### 3. [UML Diagrams Guide](file:///c:/personal%20Projects/lld/docs/uml_guide.md)
Discover how to draw and read UML diagrams. It covers class structural relationships (Inheritance, Implementation, Association, Aggregation, Composition, Dependency) and behavioral diagrams (Sequence and State diagrams) using Mermaid.
